HTML入門 - 基本的なタグを学ぼう

このページは、HTMLの基本的なタグとその使い方を学ぶための教材サンプルです。

HTML文書の基本構造

すべてのHTML文書は、次のような基本的な構造を持っています。

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>ページのタイトル</title>
</head>
<body>
    <!-- ここにページの内容を書きます -->
</body>
</html>
  • <!DOCTYPE html>: この文書がHTML5であることを宣言します。
  • <html>: HTML文書全体のルート（根っこ）要素です。lang="ja"は、このページの言語が日本語であることを示します。
  • <head>: 文書のメタ情報（文字コード、タイトルなど）を含みます。ブラウザには直接表示されません。
  • <meta charset="UTF-8">: 文字コードをUTF-8に指定します。日本語などの多言語を正しく表示するために重要です。
  • <title>: ブラウザのタブや検索結果に表示されるページのタイトルを指定します。
  • <body>: 実際にブラウザに表示されるコンテンツ（テキスト、画像、リンクなど）を記述します。

見出し (Headings) - <h1> から <h6>

見出しは、文章の構造を示すために使います。<h1>が最も大きな見出しで、<h6>が最も小さな見出しです。

例： <h1>大見出し</h1>

これはh1見出しです

これはh2見出しです

これはh3見出しです

これはh4見出しです

これはh5見出しです
これはh6見出しです

コード例:

<h1>これはh1見出しです</h1>
<h2>これはh2見出しです</h2>
<h3>これはh3見出しです</h3>

段落 (Paragraph) - <p>

文章のまとまり（段落）を作るには<p>タグを使います。

例： <p>これは段落です。</p>

これは最初の段落です。HTMLはウェブページの構造を定義するための言語です。

これは二つ目の段落です。タグを使って要素をマークアップします。

コード例:

<p>これは最初の段落です。HTMLはウェブページの構造を定義するための言語です。</p>
<p>これは二つ目の段落です。タグを使って要素をマークアップします。</p>

リンク (Link) - <a>

他のページやウェブサイトへのリンクを作成するには<a>タグを使います。href属性にリンク先のURLを指定します。

例： <a href="https://www.google.com">Googleへのリンク</a>

Googleを新しいタブで開く

ページトップへ戻る (このページ内リンク)

コード例:

<!-- 外部リンク -->
<a href="https://www.google.com" target="_blank">Googleを新しいタブで開く</a>

<!-- ページ内リンク (id="top" がページのどこかにある想定) -->
<a href="#top">ページトップへ戻る</a>

target="_blank" を追加すると、リンクが新しいタブまたはウィンドウで開かれます。

画像 (Image) - <img>

ページに画像を表示するには<img>タグを使います。このタグは閉じタグが必要ありません。

  • src属性: 表示する画像のファイルパスまたはURLを指定します。
  • alt属性: 画像が表示されない場合や、スクリーンリーダー（視覚障がい者向けの読み上げソフト）が使用された場合に表示される代替テキストを指定します。これは非常に重要です。
  • width属性 / height属性: 画像の幅や高さを指定できます（CSSでの指定が推奨されることも多いです）。

例： <img src="image.jpg" alt="説明的な代替テキスト">

コード内で画像タグを示す場合は <code>&lt;img&gt;</code> のように書きます。

以下にサンプル画像を表示します。

コード例:

<!-- ローカルの画像ファイルを表示 -->
<img src="images/sample.jpg" alt="サンプル画像の代替テキスト" width="200">

<!-- ウェブ上の画像を表示 -->
<img src="https://via.placeholder.com/150" alt="プレースホルダー画像">

リスト (List) - <ul>, <ol>, <li>

項目をリスト表示するにはリストタグを使います。

  • <ul>: 順序なしリスト (Unordered List)。通常、箇条書き（点など）で表示されます。
  • <ol>: 順序付きリスト (Ordered List)。通常、番号付きで表示されます。
  • <li>: リスト項目 (List Item)。<ul> または <ol> の中に記述します。

例： <ul><li>項目1</li><li>項目2</li></ul>

順序なしリスト (ul)

  • りんご
  • バナナ
  • オレンジ

順序付きリスト (ol)

  1. 最初にこれをやる
  2. 次にこれをやる
  3. 最後にこれをやる

コード例:

<h3>順序なしリスト (ul)</h3>
<ul>
    <li>りんご</li>
    <li>バナナ</li>
    <li>オレンジ</li>
</ul>

<h3>順序付きリスト (ol)</h3>
<ol>
    <li>最初にこれをやる</li>
    <li>次にこれをやる</li>
    <li>最後にこれをやる</li>
</ol>

強調 (Emphasis) - <strong>, <em>

テキストの一部を強調するために使います。

  • <strong>: 重要性が高い内容を示します。通常、太字で表示されます。
  • <em>: 強調したい箇所を示します (Emphasis)。通常、斜体で表示されます。

例： <p>これは <strong>重要</strong> です。そして <em>強調</em> したい部分です。</p>

この情報は非常に重要です。必ず確認してください。

この単語は特に強調したい部分です。

コード例:

<p>この情報は<strong>非常に重要</strong>です。必ず確認してください。</p>
<p>この単語は<em>特に強調</em>したい部分です。</p>

コードの表示 - <code>, <pre>

HTML内にプログラムコードやタグを表示したい場合に使います。

  • <code>: 短いコードの断片（インラインコード）を示します。
  • <pre>: 整形済みのテキストブロック（複数行のコードなど）を示します。通常、等幅フォントで表示され、スペースや改行がそのまま反映されます。<code>タグと組み合わせて使うことが多いです。

例： <p>HTMLタグは <code><p></code> のように書きます。</p>

例： <pre><code>function hello() { console.log("Hello"); }</code></pre>

注意: HTML内で < や > をそのまま表示したい場合は、&lt; (より小なり) や &gt; (より大なり) のような文字実体参照を使う必要があります。

インラインコード: <div> タグは区分を作るために使います。

複数行のコードブロック:

<!DOCTYPE html>
<html>
<head>
    <title>サンプル</title>
</head>
<body>
    <h1>こんにちは</h1>
</body>
</html>

コード例:

<p>インラインコード: <code>&lt;div&gt;</code> タグは区分を作るために使います。</p>

<p>複数行のコードブロック:</p>
<pre><code>&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head>
    &lt;title&gt;サンプル&lt;/title>
&lt;/head>
&lt;body&gt;
    &lt;h1&gt;こんにちは&lt;/h1>
&lt;/body>
&lt;/html&gt;</code></pre>

コメント - <!-- -->

HTMLコード内にメモを残したい場合や、一時的にコードの一部を無効化したい場合にコメントを使います。コメントはブラウザには表示されません。

例： <!-- これはコメントです。ブラウザには表示されません。 -->

下の行にはコメントがありますが、表示されません。

コード例:

<p>これは表示されるテキストです。</p>
<!-- これはコメントです。メモを残すのに便利です。 -->
<!-- <p>この段落はコメントアウトされているため表示されません。</p> -->

テーブル (Table) - <table>, <tr>, <th>, <td>

データを表形式で表示するにはテーブル関連のタグを使います。

  • <table>: テーブル全体を囲みます。
  • <tr>: テーブルの行 (Table Row) を定義します。
  • <th>: テーブルの見出しセル (Table Header) を定義します。通常、太字かつ中央揃えで表示されます。<tr> の中に配置します。
  • <td>: テーブルのデータセル (Table Data) を定義します。<tr> の中に配置します。

例：

<table border="1"> <!-- border属性は見た目のためですが、CSSでの装飾が推奨されます -->
  <tr>
    <th>見出し1</th>
    <th>見出し2</th>
  </tr>
  <tr>
    <td>データ1</td>
    <td>データ2</td>
  </tr>
</table>

border="1" はテーブルに枠線を表示しますが、現在ではCSSでスタイルを指定するのが一般的です。

簡単なテーブルの例

名前 年齢 職業
山田 太郎 30 エンジニア
佐藤 花子 25 デザイナー
鈴木 一郎 42 マネージャー

(注: 枠線とスタイルは例示のためインラインCSSとborder属性を使用しています)

テーブルの構造をより明確にするために <thead> (ヘッダー部分), <tbody> (本体部分), <tfoot> (フッター部分) タグを使うことも推奨されます。

コード例:

<table border="1"> <!-- CSSでの装飾推奨 -->
  <thead>
    <tr>
      <th>名前</th>
      <th>年齢</th>
      <th>職業</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>山田 太郎</td>
      <td>30</td>
      <td>エンジニア</td>
    </tr>
    <tr>
      <td>佐藤 花子</td>
      <td>25</td>
      <td>デザイナー</td>
    </tr>
  </tbody>
</table>

まとめと次のステップ

ここまでで、HTMLの基本的なタグの使い方を学びました。これらのタグを組み合わせることで、簡単なウェブページを作成できます。

  • 練習してみよう: このファイルを元に、自分で内容を書き換えたり、新しいタグを追加したりしてみましょう。
  • さらに学ぶには:
    • 他のHTMLタグ（テーブル<table>, フォーム<form>など）も学んでみましょう。
    • CSS (Cascading Style Sheets) を学ぶと、色やレイアウトなど、ページの見た目を装飾することができます。HTMLとCSSはセットで使われることが多いです。
