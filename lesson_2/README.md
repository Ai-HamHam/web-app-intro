# 準備
1. `cd lesson_2`でlesson_2フォルダに移動してください。
2. `python server.py`を実行してください。
3. 「ポート」のタブへ移動し、「転送されたアドレス」にある地球儀のアイコンをクリックしてください。
4. ブラウザが開き、`client.html`が表示されます。
