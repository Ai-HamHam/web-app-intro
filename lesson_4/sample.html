JavaScript基本文法 (初心者向け解説)

JavaScriptは、ウェブサイトに動きをつけたり、ゲームを作ったり、色々なことができる人気のプログラミング言語です。最初は難しく感じるかもしれませんが、基本的なルールから少しずつ覚えていきましょう！

1. コンピューターへの「指示」の書き方 (文)

プログラムは、コンピューターに対する「指示」の集まりです。JavaScriptでは、一つ一つの指示を「文 (Statement)」と呼びます。

  • 指示の終わりにはセミコロン ; をつけるのが基本です。（日本語の「。」のようなもの）
  • なくても動くことも多いですが、指示の区切りをはっきりさせるために、つける習慣をつけるのがおすすめです。
// 例：画面に「こんにちは！」と表示する指示
console.log("こんにちは！");

2. メモを残す方法 (コメント)

コードの中に、人間が読むためのメモを残すことができます。これを「コメント」と呼びます。コンピューターはこの部分を無視するので、プログラムの動きには影響しません。

  • 一行だけのメモ: // を書くと、その行の終わりまでがコメントになります。
  • 複数行にわたるメモ: /* と */ で囲んだ部分がコメントになります。
// これは一行コメントです。変数messageに自己紹介文を入れています。
let message = "はじめまして！";

/*
  これは複数行コメントです。
  次の行で、messageの内容を画面に表示します。
*/
console.log(message);

3. データを入れておく「箱」 (変数)

プログラムでは、数値や文字などのデータを一時的に保存しておく場所が必要です。そのための「名前付きの箱」が「変数 (Variable)」です。

  • let (レット): 中身を後から変えられる箱を作るときに使います。
  • const (コンスト): 一度入れたら中身を変えない（変えられない）箱を作るときに使います。基本的にはこちらを使うのがおすすめです。
  • var (ヴァー): 昔使われていた書き方です。今は let や const を使いましょう。
// let を使った例：年齢を入れる箱 (後で変わる可能性がある)
let age = 20;
age = 21; // 中身を21に変える

// const を使った例：名前を入れる箱 (基本的に変わらない)
const name = "田中";
// name = "鈴木"; // エラーになります！constの箱の中身は変えられません。

console.log(name); // "田中" と表示される
console.log(age);  // 21 と表示される

4. データの種類 (データ型)

変数に入れるデータには、いくつかの種類があります。よく使うものを覚えましょう。

  • 数値 (Number): 100 や 3.14 のような数字。計算に使います。
  • 文字列 (String): "こんにちは" や '山田' のような文字や文章。" または ' で囲みます。
  • 真偽値 (Boolean): true (正しい) か false (間違い) のどちらかを表す値。条件分岐（もし〜なら）などで使います。
  • オブジェクト (Object): 関連するデータをまとめて管理するのに使います。 { キー: 値 } のように書きます。（少し難しいので、最初はこういうものがある、くらいでOKです）
  • 配列 (Array): いくつかのデータを順番に並べてリストにしたもの。 [値1, 値2] のように書きます。（これもオブジェクトの一種です） 
let score = 100;                // 数値
let greeting = "おはよう";      // 文字列
let isMember = true;            // 真偽値 (true = 会員である)
let user = { name: "佐藤", age: 30 }; // オブジェクト
let colors = ["赤", "青", "黄"];    // 配列

5. 計算や比較をする記号 (演算子)

データを使って計算したり、比べたりするための記号です。

算術演算子

  • + (足し算), - (引き算), * (掛け算), / (割り算)

代入演算子

  • = (右の値を左の変数に入れる)

比較演算子 (結果は true か false になります)

  • === (左右の値と種類が同じか？ ← これを使いましょう！)
  • !== (左右の値か種類が違うか？ ← これを使いましょう！)
  • > (左が大きい), < (右が大きい)
  • >= (左が大きいか等しい), <= (右が大きいか等しい)

論理演算子 (条件を組み合わせる)

  • && (AND: かつ)
  • || (OR: または)
  • ! (NOT: ではない)
let price = 100;
let tax = 1.1;
let totalPrice = price * tax; // 掛け算 (110 になる)

let point = 85;
let isPass = (point >= 60); // 比較 (pointが60以上ならtrue)

console.log(totalPrice);  // 110 が表示される
console.log(isPass);      // true が表示される

6. 条件によって動きを変える (if文)

「もし〜だったら、Aをする。そうでなければ、Bをする」というように、条件によって処理を変えたいときに使います。

let hour = 14; // 今の時刻 (14時)

if (hour < 12) {
  console.log("午前中です");
} else if (hour < 18) {
  console.log("午後です"); // この条件に合うので、これが表示される
} else {
  console.log("夜です");
}

7. 同じことを繰り返す (ループ)

同じような処理を何度も繰り返したいときに使います。forループは決まった回数繰り返すときによく使います。

// 0から4まで、5回繰り返す
for (let i = 0; i < 5; i++) {
  console.log("現在 " + i + " 回目です");
}

8. 処理をまとめた「道具」 (関数)

一連の処理に名前をつけて、何度も呼び出して使えるようにしたものです。「関数 (Function)」と呼びます。同じコードを何度も書かなくて済むので便利です。

// 「挨拶する」という処理をまとめた関数
function sayHello(name) {
  console.log("こんにちは、" + name + "さん！");
}

// 関数を使う（呼び出す）
sayHello("鈴木"); // "こんにちは、鈴木さん！" と表示される
sayHello("加藤"); // "こんにちは、加藤さん！" と表示される

まとめ

まずはこれらの基本的なルールを、実際にコードを書いて試してみるのが一番です。ブラウザの「開発者ツール」や、Node.jsという環境、オンラインのJavaScript実行環境などで手軽に試せます。

最初はエラーが出たり、うまくいかなかったりするかもしれませんが、少しずつ慣れていくので安心してください。楽しんで学ぶことが大切です！
